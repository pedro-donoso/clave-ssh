## Clave SSH

1. Crear clave SSH -> desde terminal:

```
ssh-keygen

```
  *elegir contraseña*

![keygen](https://user-images.githubusercontent.com/68760595/134391907-ee2869a5-fef3-4225-9a45-565e1cdc9838.png)

2. Acceder a SSH (desde raíz):

```
cd .ssh

```
```
ls

```
 *aparece clave privada:*
 
 id_rsa
 
 *y clave pública:*
 
 id_rsa.pub
